# Swad Odisha - Authentic Flavors of Tradition

[![React](https://img.shields.io/badge/React-18.3-blue.svg)](https://reactjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue.svg)](https://www.typescriptlang.org/)
[![Vite](https://img.shields.io/badge/Vite-6.x-646CFF.svg)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.x-38B2AC.svg)](https://tailwindcss.com/)

Swad Odisha is a premium web platform dedicated to showcasing and delivering the authentic, rich cultural heritage of Odia cuisine. From the iconic Pakhala Bhata to the world-renowned Chhena Poda, experience the true taste of Odisha with a modern, interactive interface.

## ✨ Features

- **🚀 Modern Tech Stack:** Built with React 18, TypeScript, and Vite for blazing fast performance.
- **🎨 Immersive UI/UX:** Features advanced animations using **Framer Motion** and interactive 3D effects with **React Parallax Tilt**.
- **🍱 Extensive Menu:** Browse through categorized traditional dishes with dietary indicators (Veg/Spicy).
- **🛒 Seamless Cart Experience:** Real-time cart management using React Context API.
- **📱 Fully Responsive:** Optimized for all devices, from mobile to desktop.
- **🏗️ Architectural Excellence:** Strict TypeScript implementation for robust and maintainable code.

## 🛠️ Technology Stack

- **Frontend:** React.js, TypeScript
- **Styling:** Tailwind CSS, Shadcn UI
- **Animations:** Framer Motion, React Parallax Tilt
- **Build Tool:** Vite
- **Icons:** Lucide React

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Himansu-Nayak/Swad-Odisha.git
   cd Swad-Odisha
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
